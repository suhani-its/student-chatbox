# student-chatbox
AI-powered Student Support Chatbot built with Flask, Gemini AI and SQLite. Helps students with college queries, schedules, and resources.
 Project Explanation
 This is a **Student Support Chatbot** built using Flask and Gemini AI. It helps students get instant answers about college schedules, fees, exam results, and   other queries. The chatbot uses AI to understand questions and provide accurate responses.

     - **How it works**: Students type their queries, AI processes them, and replies with relevant info.
     - **Tech used**: Flask (backend), Gemini AI (NLP), SQLite (database)
     - **Purpose**: Reduce workload on college admins by answering common student questions automatically.
